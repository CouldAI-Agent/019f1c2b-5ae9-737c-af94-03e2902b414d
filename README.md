# Gold Inventory App

A Flutter application for managing a gold inventory with categories.

## Features
- View gold inventory categories (e.g., Rings, Necklaces, Coins, Bars)
- Track individual items with details such as weight and purity (e.g., 22k, 24k)
- Responsive design for mobile and desktop

## Tech Stack
- Flutter
- Material Design 3

## Setup and Run
1. Ensure you have Flutter installed.
2. Run `flutter pub get` to fetch dependencies.
3. Run `flutter run` to start the application.
